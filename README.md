# Algorithm Visualizer

An interactive algorithm visualization tool built with HTML, CSS, and JavaScript.

## Live Demo
[View Live](https://lexogamer786-cyber.github.io/algorithm-visualizer/)

## Algorithms Included

### Sorting
- **Bubble Sort** — compares adjacent elements and swaps
- **Selection Sort** — finds minimum and places in position
- **Insertion Sort** — builds sorted array one element at a time
- **Quick Sort** — divide and conquer with pivot element

### Searching
- **Linear Search** — checks each element one by one
- **Binary Search** — divides sorted array in half each time

## Features
- Real-time animated bar visualization
- Color coded states — comparing, swapping, pivot, sorted, found
- Live stats — comparisons and swaps counter
- Speed control — 1x to 10x
- Array size control — 10 to 60 elements
- Step by step log panel
- Custom search target input
- Mobile responsive

## What I Used
- HTML5 — structure
- CSS3 — animations and styling
- JavaScript — all algorithm logic and DOM manipulation
